Two Sum Easy

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution...

Google Amazon
1def twoSum(nums, target):
2 # Use hashmap for lookup
3 seen = {}
4 for i, num in enumerate(nums):
5 diff = target - num
6 if diff in seen:
7 return [seen[diff], i]
8 seen[num] = i
9 return []
✨ Gemini AI Assistant
What is the space complexity of this solution?
The space complexity is O(N), where N is the number of elements in nums. This is because, in the worst case, we store each element and its index in the hash map.
